# Local Diagnostic Queries

These queries are retained for GuardLint development, debugging, and ad hoc
inventory checks. They are not part of the supported paper/artifact query set.
The authoritative supported queries live one directory up and are:

- `missing_guards.ql`
- `missing_guard_detail.ql`
- `good_guards.ql`
- `good_guard_detail.ql`
- `redundant_guards.ql`
- `redundant_guard_detail.ql`
